To get started, take a look at src/app/page.tsx.

### Environment Variables

For the Genkit AI integration to work, you'll need to set the following environment variable in your deployment platform:

```
GOOGLE_GENAI_API_KEY=your_google_ai_api_key_here
```

You can get your Google AI API key from the [Google AI Studio](https://aistudio.google.com/).
